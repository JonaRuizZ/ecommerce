1.2 - Creación del proyecto y del repositorio
 - Usaremos vite que es la alternativa a webpack porque vite está más optimizado, es mas rápido, mas ligero etc.
 - En la terminal (cva: Create Vite App): 
 > npx cva ecommerce 
 > React
 > JavaScript

  - Después de haber creado la aplicación debemos irnos al git bash:
1) $ cd c/Users/jonat/Escritorio/Proyectos/proyectos-react/ecommerce
2) $ yarn

 - Ahora levantamos el proyecto en el navegador
 1) puede ser en la git bash con yarn:
 $ yarn dev
 2) O puede ser en la terminal de vscode con:
 PS C:\Users\jonat\Escritorio\Proyectos\proyectos-react\ecommerce> npm run dev

  - ¿Qué archivos podemos eliminar ya que no los vamos a necesitar?
1) Logo de vite ubicado en public/vite.svg
2) Dentro de src App.css, index.css, y la carpeta assets
3) Limpiamos el código de App.jsx y el main.jsx

 - Tenemos que crear un repositorio con git.

 - En el archivo .gitignore debemos agregar una linea:
 *.lock 
 Significa que no suba los archivos .lock porque en ella estan todas las dependencias
 
 - Creamos un archivo README.md (.md = Markdown)
